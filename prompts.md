# Prompts Used
## 1. Project Setup

Created a full-stack MERN project using React, Vite, Node.js, Express, MongoDB, JWT Authentication, and Stripe Test Mode. Generated a clean folder structure suitable for a student-level full-stack application.

## 2. UI Development

Used AI to generate React components based on the provided Figma design, including:

Authentication pages
Dashboard
Sidebar
Header
Task cards
Kanban board
Statistics cards
Analytics section
Responsive layouts

The generated UI was customized and adjusted to match the required design.

## 3. Backend Development

Generated Express server structure with:

MongoDB connection
Mongoose models
Authentication middleware
Controllers
API routes
JWT implementation
Error handling
## 4. CRUD Functionality

Implemented complete CRUD operations for tasks:

Create Task
Read Tasks
Update Task
Delete Task

Integrated the frontend with backend REST APIs using Axios.

## 5. Authentication

Implemented:

User Registration
User Login
JWT Authentication
Protected Routes
Token validation
Logout functionality
## 6. Data Ownership

Implemented backend validation to ensure users can only access and manage their own tasks by comparing the authenticated user's ID with the task owner's ID.

## 7. Dashboard Analytics

Used AI guidance to integrate Recharts for dynamic analytics including:

Tasks by Status
Tasks by Priority
Completed vs Pending Tasks
Weekly Task Overview

Charts update automatically based on the latest task data.

## 8. Stripe Integration

Implemented Stripe Checkout in Test Mode with:

Checkout Session API
Upgrade to Pro button
Success page
Cancel page

Environment variables were used to keep API keys secure.

## 9. Error Handling

Added validation and error handling for:

Invalid input
Authentication failures
Authorization failures
Missing resources
Server errors

Appropriate HTTP status codes were returned from the backend.
